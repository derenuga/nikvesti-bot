# Мітки черг відключень (energy_outage_labels.json)

Файл `handlers/energy_outage_labels.json` — словник `"назва черги" → [список локацій]`.  
Використовується в `build_message()` у `handlers/energy_outage.py`:  
під рядком черги виводиться курсивна підказка з мікрорайонами/дільницями.

---

## Джерела даних

### 1. CSV-адресна база (idfilial=15, команда `/outage_export`)
Файл `outage_streets_filial_15_1.csv` — всі вулиці Миколаєва з прив'язкою до черги.  
Геокодування → кластеризація → ручна мітка мікрорайону дала `queue_clusters_mikroraion_15.json`.

**Обмеження**: у адресній базі Миколаєва частина вулиць прив'язана до черг ГАВ (1–10),  
а не ГПВ (1.1–6.2). Тому для черг ГПВ **1.1, 1.2, 3.1, 3.2, 5.1, 5.2** міських мікрорайонів  
у базі немає — використовуються назви обласних дільниць із PDF.

Черги ГПВ, що мають міські мікрорайони в базі: **2.1, 2.2, 4.1, 4.2, 6.1, 6.2**.

### 2. PDF обленерго "Черги ГПВ" (Chergi_GPV.pdf)
Офіційний документ із розбивкою всіх 12 черг ГПВ по обласних РЕМ-дільницях.  
Використовується для черг без міських адрес, і для доповнення обласним контекстом.

## Формат повідомлення `/outage`

```
⚡ Графік відключень на 1 липня 2026

📋 ГПВ:
Черга 3.1: немає світла 2 год — 19:00–21:00
<i>Центр міста Миколаєва, Баштанська, Березнегуватська, Варварівська та ін.</i>

Черга 4.1: немає світла 2 год — 17:00–19:00; можливо — 16:30–17:00
<i>Абісінія, Ліски, Намив, Слобідка та ін.</i>
```

- `parse_mode="HTML"` у Telegram
- Години рахуються тільки по `OFF` слотах (не `PROBABLY_OFF`)
- Проміжки часу через ` / `
- `MAX_SHOWN = 5` локацій + `" та ін."` якщо більше

## Як розширити labels.json

1. Дізнатись нові черги через `/api/outage-queue/by-type/{type_id}`
2. Для ГПВ — перевірити PDF обленерго (оновлюється рідко)
3. Для міських адрес — запустити `/outage_export 15`, потім геокод-скрипт, потім ручна мітка кластерів
4. Додати до `handlers/energy_outage_labels.json` ключ = назва черги (рядок), значення = масив локацій

**Примітка**: для ГАВ-черг (1–10, (Р)) labels.json містить мікрорайони на основі адресної бази.  
Для ГПВ-черг без адресної прив'язки — обласні дільниці з PDF.
