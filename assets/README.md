# assets

Статичні файли бренду для графіків Лиса.

- `nikvesti_logo.png` — логотип МикВісті (бажано з прозорим фоном). Якщо файл
  присутній, `handlers/query_router.render_chart` ставить його напівпрозорим
  вотермарком у правий верхній кут кожного графіка. Якщо файлу немає — графік
  малюється без логотипа, без помилки (див. `_stamp_logo`).
